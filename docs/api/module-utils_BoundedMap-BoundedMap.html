new BoundedMap()

Map with maximum size limit and LRU eviction policy When the map exceeds maxSize, the least recently used entry is removed.

Source:

Example
const cache = new BoundedMap(100);
cache.set('key1', 'value1');
const value = cache.get('key1'); // Marks as recently used
